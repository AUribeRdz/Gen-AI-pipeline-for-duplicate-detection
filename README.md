# Gen-AI pipeline for duplicate detection Enterprise product catalogs accumulate thousands of duplicate records over time, silently degrading search, recommendations, and inventory accuracy. This project applies Generative AI to detect and resolve those duplicates at scale. ## What it does - Embeds product records using sentence-transformer models - Computes vector similarity to surface candidate duplicate pairs - Uses an LLM (GPT-4o / configurable) to make a final merge/keep decision with reasoning - Outputs a resolution report with confidence scores and audit trail ## Why it matters for retail Duplicate SKUs affect pricing consistency, search ranking, and downstream ML model quality. This pipeline addresses the root cause rather than patching symptoms. ## Tech stack Python · LLM API (OpenAI-compatible) · Sentence Transformers · Pandas · Jupyter Notebook ## How to run ```bash pip install -r requirements.txt jupyter notebook duplicate_detection_pipeline.ipynb ``` ## Skills demonstrated Generative AI · RAG-adjacent retrieval · Prompt engineering · Data quality · Enterprise data managemen
